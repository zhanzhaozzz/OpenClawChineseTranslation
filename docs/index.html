4 步上手

需要 Node.js >= 22（下载），检查版本：node -v

1

安装

npm 全局安装汉化版

npm i -g @qingchencloud/openclaw-zh@latest
2

初始化

运行向导配置 AI 模型

openclaw onboard --install-daemon
3

启动网关

后台运行服务

openclaw gateway
4

打开控制台

浏览器自动打开中文界面

openclaw dashboard

安装方式

三种方式任选，脚本安装最省心

Windows PowerShell
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/gh/1186258278/OpenClawChineseTranslation@main/install.ps1" -OutFile "install.ps1"; .\install.ps1
macOS / Linux
curl -fsSL -o install.sh https://cdn.jsdelivr.net/gh/1186258278/OpenClawChineseTranslation@main/install.sh && bash install.sh

开源脚本，可在 GitHub 审查源码

npm 全局安装
npm install -g @qingchencloud/openclaw-zh@latest
安装后运行
openclaw onboard
一键部署脚本（推荐，自动配置远程访问）
curl -fsSL https://cdn.jsdelivr.net/gh/1186258278/OpenClawChineseTranslation@main/docker-deploy.sh | bash
Docker Compose
curl -fsSL https://cdn.jsdelivr.net/gh/1186258278/OpenClawChineseTranslation@main/docker-compose.yml -o docker-compose.yml && docker-compose up -d

脚本/npm 需要 Node.js >= 22.12.0 | Docker 需要 Docker Engine | 查看所有版本
